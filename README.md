# Sort CSS Property

gulpで動きます。

## 環境

```bash
$ node -v
v5.2.0

$ npm -v
3.5.2
```

## 構築手順

```bash
$ npm install -g gulp
$ git clone git@github.com:kzsnkgw/sort_css_property.git
$ cd sort_css_property
$ npm install
$ gulp default
```

### 概要

[csscomb](http://csscomb.com/)を参照。
ソートするルールは[csscomb.json](https://github.com/kzsnkgw/sort_css_property/blob/master/.csscomb.json)で設定します。
